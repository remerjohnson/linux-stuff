# Some tips 

A good place to put bash scripts is /usr/bin or ~/bin  
For simple aliases, /etc/bash.bashrc or try ~/.bash_aliases  

To switch from a default shell to another:
  
```
exec bash   # switches to bash  
exec zsh    # switches to zsh  
```
